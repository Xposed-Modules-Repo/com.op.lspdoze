# LSPDoze

用于 Android 设备息屏续航优化。

## 功能介绍

### DeepDoze（深度 Doze）
- 息屏后自动进入深度 Doze 模式，降低待机功耗
- 默认延迟 60 秒，可自定义
- 内置状态检测机制（1/5/30/60 分钟递增），异常退出后自动恢复

### DeepSleep（深度睡眠）
- 息屏后进入 OPPO 深度睡眠模式，省电就完了
- 仅支持 OPPO/OnePlus/Realme 设备
- 自动拦截系统退出深度睡眠的尝试
- 智能场景检测：通话、导航、热点、USB 共享、投屏、数据传输 等场景下自动跳过

### AOD（常显息屏）
- 强制开启常显全屏 AOD
- 仅支持 OPPO/OnePlus/Realme 设备（Android 16+）

## LSP 作用域

| 功能        | 作用域         |
|-----------|-------------|
| DeepDoze  | 系统框架        |
| DeepSleep | 电池   |
| AOD       | 息屏 + 系统界面 |

## 配置项

通过软件UI界面配置

## 兼容性

- 最低支持：Android 11（API 30）
- 目标版本：Android 16（API 36）
- 已测试设备：一加 13、一加 9R、一加 ACE5 至尊版、部分小米和三星设备
